The Tap Master

  Github

The Tap Master is an online web game inspired by Taiko no Tatsujin(The Taiko Master). First, players are asked to select a level of difficulties. The game will start will "taikos" falling from the top of the screen. As those taikos reach to the bottom empty slots, players need to tap the taiko within an acceptable timing in order to get the scores. There are three levels of game and each levels comes with different speed as welll as different music. What's the highest score you can get?
